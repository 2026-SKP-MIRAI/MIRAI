# resume

> 자소서 업로드 및 면접 시작 플로우

## 목적

PDF 자소서를 업로드하고 면접 모드를 선택하여 면접 세션을 시작하는 페이지.

## 구조

```
resume/
├── page.tsx   # Client Component — 업로드 폼 + 모드 선택
└── .ai.md     # 이 파일
```

## 역할

- 기본 흐름: PDF 업로드 → `POST /api/resume/questions` → 질문 생성 → 모드 선택 → 면접 시작
- **재사용 흐름** (이슈 #157): `?resumeId={id}` 파라미터 수신 시
  - 업로드 폼 없이 바로 `state='done'`으로 진입
  - 기존 자소서로 면접 모드 선택 화면 표시
  - 대시보드의 "이 자소서로 다시 면접하기" 버튼에서 이 경로로 진입

## 상태 흐름

```
idle → uploading → done
                     ↑
?resumeId= 있으면 바로 여기서 시작
```

## 관련 파일

- `src/app/dashboard/page.tsx` — "이 자소서로 다시 면접하기" 버튼 → `/resume?resumeId=`
- `src/app/api/resume/questions/route.ts` — POST 핸들러 (업로드 + 질문 생성)
