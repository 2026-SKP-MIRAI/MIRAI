# services/lww/ — lww

lww의 개인 서비스. 자소서 기반 모의면접 시스템을 end-to-end로 구현.

## 구조
```
lww/
└── (자율 설계)
    예: domain/      도메인 모델
        api/         백엔드 API
        frontend/    프론트엔드
        tests/       테스트
```

## 역할
- engine/ 만 호출 — LLM·파서 직접 구현 금지
- Week 1: MVP end-to-end 작동
- Week 2~3: 기획서 기능 추가
- Week 4: 특색 기능 1개 이상 + 배포 URL 제출
