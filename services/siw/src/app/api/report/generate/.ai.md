# POST /api/report/generate

## 목적
면접 세션 기반 AI 리포트를 생성하고 DB에 저장한다.

## 핵심 동작
- 인증 필요 (Supabase)
- sessionComplete 체크: false이면 400 반환
- 캐시: sessionComplete && reportJson 있으면 즉시 반환
- history < 5이면 422 반환
- engine /api/report/generate 호출 후 saveWithRetry()로 저장 (await 보장)
