# services/siw/ — 성시우

## 목적
MVP 01 완료 — 자소서 업로드 → 질문 생성 end-to-end.
DDD (Domain-Driven Design) + TDD 적용.

## 구조
```
siw/
├── src/
│   ├── app/
│   │   ├── layout.tsx                    # RootLayout (lang="ko")

│   │   └── api/resume/questions/route.ts # POST → 엔진 프록시
│   ├── components/

    ├── setup.ts
```

## 역할
- 기능 흐름: 자소서 업로드 → 질문 생성 → (Week 2~) 면접 시뮬레이션 → 역량 평가
- engine/ 만 호출 — LLM·파서 직접 구현 금지
- ENGINE_BASE_URL 환경변수로 엔진 접근 (기본값: http://localhost:8000)
- Week 1: MVP 완료 (자소서 → 질문 생성 end-to-end 작동)
- Week 2~3: 기획서 기능 2~4 (꼬리질문, 페르소나, 역량 평가) 추가
- Week 4: 특색 기능 1개 이상 + 배포 URL 제출

## 기술 스택
- Next.js 15, React 19, TypeScript (strict)
- Vitest 2, @testing-library/react 16
- 테스트: 총 20개 (API 13개 node 환경 + UI 7개 jsdom 환경)
