# services/kwan/ — 김관우

## 목적
김관우의 개인 서비스. 자소서 기반 모의면접 시스템을 end-to-end로 구현.
DDD (Domain-Driven Design) + TDD 적용.

## 현재 진행 상태
- **Week 1 MVP 완료** (2026-03-09): 자소서 PDF 업로드 → 질문 생성 end-to-end 작동

## 구조
```
kwan/
├── src/
│   ├── app/
│   │   ├── layout.tsx                        메타데이터, 폰트
```

## 역할
- 기능 흐름: 자소서 업로드 → 질문 생성 → (Week 2) 면접 시뮬레이션 → (Week 3) 역량 평가
- engine/ 만 호출 — LLM·파서 직접 구현 금지 (HTTP: ENGINE_BASE_URL)
- **Week 1 (완료)**: MVP — 자소서 → 질문 생성 end-to-end
- Week 2~3: 기획서 기능 2~4 (꼬리질문, 페르소나, 역량 평가) 추가
- Week 4: 특색 기능 1개 이상 + 배포 URL 제출

## 불변식
```
1. import anthropic, import fitz 금지 — 엔진만 LLM/파서 사용
2. ENGINE_BASE_URL 환경변수로 엔진 통신 (타임아웃 30초)
3. 에러 메시지는 한국어로 사용자 노출
4. 테스트 없는 PR 금지 (pnpm test 통과 필수)
```

## 개발 커맨드
```bash
pnpm dev          # localhost:3000
pnpm test         # Vitest 단위 테스트 (15개)
pnpm build        # 프로덕션 빌드
```
