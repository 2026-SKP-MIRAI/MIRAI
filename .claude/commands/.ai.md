# .claude/commands/

## 목적
Claude Code 슬래시 커맨드 파일 모음. 이슈 워크플로우 자동화 커맨드와 단축 별칭을 포함한다.

## 파일 목록

### 워크플로우 커맨드
- `backlog-issue.md` — 새 이슈를 Backlog에 생성
- `start-issue.md`   — 이슈 작업 시작 (워크트리/브랜치/work 폴더 생성, 이슈 assign)
- `finish-issue.md`  — 구현 완료 후 커밋 및 PR 생성
- `cleanup-issue.md` — PR 머지 후 워크트리/브랜치 정리

### 단축 별칭 (symlink)
- `bi.md` → `backlog-issue.md`
- `si.md` → `start-issue.md`
- `fi.md` → `finish-issue.md`
- `ci.md` → `cleanup-issue.md`

## 규칙
- 커맨드 파일은 YAML frontmatter의 `description` 필드를 반드시 포함한다
- 단축 별칭은 심볼릭 링크로 구현되어 원본 변경 시 자동 반영된다
