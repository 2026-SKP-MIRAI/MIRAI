# engine/docs/

## 목적
엔진 설계 문서 보관. 서비스 개발자가 엔진 API 계약을 확인하는 곳.

## 구조
```
engine/docs/
└── (향후 설계 문서 보관 — 현재 비어 있음)
```

## 역할
- 엔진 API 계약의 단일 출처는 `engine/.ai.md` (INTERFACE.md 미사용)
- 서비스 개발자는 `engine/.ai.md`에서 엔진 사용법·예외 계층·타입 확인
- 엔진 내부 구현 변경 시 `engine/.ai.md` 업데이트 필수

## 규칙 변경 시 컨펌 필요
이 파일에 기재된 규칙과 충돌하는 변경을 하려는 경우, 작업을 진행하기 전에 반드시 사람(팀원)에게 먼저 확인을 받아야 한다.
