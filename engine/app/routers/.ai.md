# engine/app/routers/

## 목적
FastAPI API 엔드포인트 정의. `/resume`, `/interview` 등 라우트를 기능별로 분리.

## 구조
```
routers/
└── (기능별 라우터 파일)
    예: resume.py      자소서 업로드·파싱 엔드포인트
        interview.py   면접 질문 생성 엔드포인트
```

## 역할
- 요청 수신 → parsers/services 호출 → 응답 반환
- 인증 로직 포함 금지 (서비스 게이트웨이 패턴 — Next.js가 인증 담당)
- 타입은 `schemas.py`의 Pydantic 모델 사용

## 작업 전 확인
- `engine/.ai.md` — API 계약 확인
- `schemas.py` — 요청·응답 타입 확인

## 규칙 변경 시 컨펌 필요
이 파일에 기재된 규칙과 충돌하는 변경을 하려는 경우, 작업을 진행하기 전에 반드시 사람(팀원)에게 먼저 확인을 받아야 한다.
