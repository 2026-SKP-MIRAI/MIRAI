## 2026년 3월 2일 주차 (Mar 2~8)

### ✨ 새 기능
- **면접 질문 자동 생성 MVP 완성** (#33): 자소서 PDF를 업로드하면 AI가 맞춤 면접 질문을 생성합니다. 엔진과 lww 서비스가 end-to-end로 연결되었습니다.
- **이슈 워크플로우 슬래시 커맨드 4종 도입** (#1, #3, #7): `/si`, `/fi`, `/bi`, `/ci` 명령으로 이슈 시작부터 PR 생성·정리까지 한 번에 처리할 수 있습니다.
- **finish-issue PR 자동화** (#7): PR 생성 시 이슈 배경 요약, 작업 내역, AC 체크 결과가 본문에 자동으로 포함됩니다.

### 🔧 개선
- **LLM 공급자 전환**: Anthropic SDK에서 OpenRouter + Gemini 2.5 Flash로 전환해 더 유연한 모델 선택이 가능해졌습니다. 설정은 `OPENROUTER_API_KEY`와 `OPENROUTER_MODEL` 환경변수로 관리됩니다.
- **워크트리 `.gitignore` 자동 연결** (#34): 새 이슈 작업 시작 시 `.gitignore`가 자동으로 심볼릭 링크되어 민감 파일이 실수로 커밋되는 일을 방지합니다.
- **엔진 구조 재정비** (#23): `app/` 계층 도입으로 라우터·서비스·파서·설정이 명확하게 분리되었습니다. 모노레포 구조로 롤백해 개발 편의성을 높였습니다.
- **커밋 컨벤션 정립** (#20): 이슈 단위 1커밋 원칙이 수립되었습니다.

### 🛡️ 보안 / 코드 품질
- **pre-commit 가드 강화**: TypeScript 파일의 LLM 직접 호출, 엔진 인증 로직 혼입, PDF/CSV 등 금지 파일 커밋을 커밋 시점에 자동 차단합니다.
- **스테이징 전 파일 검사**: `finish-issue` 워크플로우가 커밋 전 금지 파일(`.pdf`, `.csv`, `tests/fixtures/` 등)을 감지해 경고를 표시합니다.

- **Claude Code 보안 설정 강화**: deny 규칙, 권한 설정, 플러그인이 추가되었습니다.
- **TDD MVP 테스트 픽스처 준비** (#28): 엔진 예외 계약과 스펙이 정비되었고, 테스트 기반 개발 환경이 구축되었습니다.
