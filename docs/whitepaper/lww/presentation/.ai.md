# lww 발표용 PPTX + 슬라이드 이미지 생성 시스템

## 목적

lww(Fint) 서비스의 발표용 PPTX와 슬라이드별 시각화 이미지를 자동 생성한다.

## 구조

```
presentation/
├── content/
│   └── slides-data.json       # 12슬라이드 텍스트·나레이션·메타데이터
├── scripts/
│   ├── generate_pptx.py       # PPTX 생성 (python-pptx)
│   └── generate_images.py     # 슬라이드별 PNG 생성 (PIL/Pillow)
├── slides/
│   └── lww-pitch-deck.pptx    # 최종 PPTX 산출물
├── images/
│   └── slide-{01~12}.png      # 슬라이드별 시각화 이미지 (600×800)
└── slide-exports/
    └── slide-{01~12}.png      # PPTX→PDF→PNG 내보내기 (검수용)
```

## 슬라이드 구성 (12장)

| id | 타입 | 레이아웃 | 이미지 |
|----|------|----------|--------|
| 01 | 표지 | 전폭 Teal 배경 | 없음 |
| 02 | 문제 | 좌 텍스트 + 우 PIL | ① 3개 통증 카드 |
| 03 | 솔루션 | 좌 텍스트 + 우 PIL | 3축 키워드 박스 |
| 04 | Aha Moment | 좌 텍스트 + 우 PIL | 피드백 카드 UI |
| 05 | 시장 규모 | 좌 텍스트 + 우 PIL | TAM/SAM/SOM 피라미드 |
| 06 | 경쟁사 | 좌 텍스트 + 우 PIL | 2×2 포지셔닝 매트릭스 |
| 07 | 차별점 | 전폭 텍스트 | 없음 |

## 실행 방법

```bash
cd docs/whitepaper/lww/presentation

# 이미지 생성 (PIL placeholder)
python scripts/generate_images.py --data content/slides-data.json --output-dir images/ --placeholder-only

# PPTX 생성 (이미지 없음)
python scripts/generate_pptx.py --data content/slides-data.json --output slides/lww-pitch-deck.pptx

# PPTX 생성 (이미지 포함)
python scripts/generate_pptx.py --data content/slides-data.json --output slides/lww-pitch-deck.pptx --with-images

# PNG 내보내기 (검수용)
libreoffice --headless --convert-to pdf slides/lww-pitch-deck.pptx --outdir slide-exports/
pdftoppm -r 150 -png slide-exports/lww-pitch-deck.pdf slide-exports/slide
```

## 브랜드 컬러

- TEAL: `#0D9488`
- TEAL_LIGHT: `#F0FDFA`
- DARK: `#1A1A1A`
- GREY: `#6B7280`

## 이미지 생성 정책

- `NO_IMAGE_SLIDES = {7, 9}`: 이미지 없이 전폭 텍스트
- `slide 01, 12`: 커버/CTA 슬라이드 — 이미지 영역 없음
- `slide 02~11` (07, 09 제외): 우측 600×800 PNG 이미지 삽입
- PIL dispatcher 패턴: `_SLIDE_DISPATCHER[slide_id](draw, fonts)`
