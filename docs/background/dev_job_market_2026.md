### 전체 분위기 (2025~2026)

**채용 시장 구조 (사람인+점핏 2025 상반기 개발자 채용 리포트 기준):**

| 항목 | 수치 |
|------|------|
| 전체 채용공고 표본 | **10만 건** |
| 백엔드 비중 | **16.2%** (전체 직무 중 1위) |
| 프론트엔드 비중 | **11.1%** (전체 직무 중 3위) |
| 신입 공고 비중 | **0.8%** (전체 중 신입) |
| 기업이 가장 원하는 경력 | 4~7년차 49.7%, 1~3년차 19.6%, 신입 12.4% |

> 출처: [사람인+점핏 2025 상반기 개발자 채용 리포트](https://www.saraminhr.co.kr/pr/press_view.do?id=218)

**결론:** "개발자 빙하기"는 **신입 한정**. 경력직(특히 AI 역량 보유)은 수요 지속.

## 시사점

### MirAI 팀 멘티들에게 주는 시사점

#### 1. 신입 공채보다 포트폴리오 + AI 활용 데모가 더 중요

```
기존 취업 경로: 코딩테스트 → 공채 지원
현재 현실:      포트폴리오 + AI 활용 데모 → 스타트업/소규모 기업 지원
```

- 공채로 신입 뽑는 대기업은 극소수 (한국 잡코리아 기준 신입 비중 5~10%)
- **"실제로 쓰이는 제품을 만든 경험"** = 가장 강력한 차별점
- MirAI = 포트폴리오의 핵심 소재

#### 2. AI 역량이 연봉과 채용 모두에서 프리미엄

- 한국: AI 역량 보유 시 ML/AI 공고 프리미엄 (ML 2,199건 vs DS 신입 11건)
- 미국: ML/AI Engineer 평균 TC $242,850 vs SWE 중앙값 $200K
- **Claude Code 활용 + LLM API 통합 경험** = 즉시 어필 가능한 스킬

#### 3. 어떤 개발자가 되느냐가 핵심

| 구분 | 시장 전망 | 전략 |
|------|---------|------|
| AI 없는 일반 개발자 | ❄️ 수요 감소 | 전환 필요 |
| AI 활용 개발자 | 🟢 수요 유지 | 현재 준비 중 |
| AI 환경 설계 개발자 (Harness) | 🔥 최고 수요 | MirAI로 실습 중 |

#### 4. 직군별 현실적 취업 목표

| 멘티 | 직군 | 현실적 첫 취업 타겟 | 차별점 |
|------|------|------------------|--------|
| 성시우 | AI-native PM/Builder | 스타트업 프로덕트 빌더 or PM | MirAI 기획+데모 포트폴리오 |
| 김관우 | AI-first DevOps | 스타트업 인프라/DevOps | CI/CD + 에이전트 검증 인프라 경험 |
| 유동선 | Technical Founder | 스타트업 창업 or 초기멤버 | 도메인 전문성 + AI 파이프라인 |
| 이승현 | AI-native Backend | AI 서비스 백엔드 엔지니어 | LLM API 연동 FastAPI 경험 |

#### 5. 해외 취업을 고려한다면

- ML/AI 엔지니어가 한국-해외 연봉 격차 가장 큼 (한국 ~$40K vs 미국 $242,850 TC 중앙값)
- **"LLM 기반 서비스 백엔드"** 포지션이 해외 취업 현실적 진입점
- GitHub + 영어 README + 데모 URL = 해외 취업 기본 조건
