# AI 모의면접 서비스 경쟁사 분석 2026

> 작성: 2026-03-01 / 출처: 팀원 직접 조사 (Notion) + 웹 검색 보완
> 조사 참여: 성시우(잡다), 김관우(뷰인터), 이승현(몬스터) / 멘토(사람인)
> 연계 문서: `docs/whitepaper/MirAI_proposal.md` §1 시장 분석

## 선요약

| 서비스 | 포지셔닝 | MirAI 기회 |
|--------|----------|-----------|
| **잡다** | B2C 역량검사 + AI 채용 매칭. 면접 연습보다 역량 측정이 주목적 | 연습·코칭 경험 부재 |
| **뷰인터** | B2B 채용 자동화. 정량 지표로 면접 비용 절감 | 지원자 경험 최악. 피드백 없음 |
| **몬스터** | B2B 채용 검증. 팩트체크+멀티모달 분석. 고압적 환경 | 지원자 성장 0. "평가" vs "연습" |
| **사람인 AI면접** | B2C 모의면접. 6명 페르소나. 꼬리질문+피드백 | 가장 근접한 경쟁자. 差別點 필요 |
| **MirAI 방향** | 자소서 기반 맞춤 질문 + 실행형 피드백 + 코칭 경험 | 지원자 편 서비스 |

**핵심 인사이트:** 기존 B2B 서비스(뷰인터·몬스터)는 기업이 클라이언트 — 지원자 경험은 후순위. MirAI는 **지원자가 클라이언트**인 B2C 모의면접+코칭으로 공백 점유 가능.

## 1. 잡다 (Jobda)

> 조사: 성시우 / 직접 UX 체험 기반

### 기본 정보

- **운영사**: 미상 (잡다 자체 플랫폼)
- **타겟**: B2C 구직자, 특히 신입
- **과금 모델**: 월 5회 무료 응시 (기업이 채용 연계 시 추가 가능)
- **규모**: 750개+ 기업 제휴, 40만+ 사용자 (2025 기준)

### 핵심 기능

| 기능 | 내용 |
|------|------|
| AI 역량검사 | 게임형 역량측정 + 영상 면접 통합 |
| 온보딩 흐름 | 로그인 → 직군·직무 선택 → 경력 여부 → 성향 검사 → AI 면접 |
| 성향 검사 | 개인 성향·직무 적합성 사전 측정 |
| 역량 기반 매칭 | 측정 결과 기반 기업이 먼저 면접 제안 (역전 채용) |
| 개인정보 관리 | 인적사항 저장 → 반복 지원 편의 |

### UX 관찰 (성시우)

- 직무 선택 → 성향 검사 → AI 면접 온보딩이 즉각적으로 시작됨 (마찰 최소화)
- 대부분의 콘텐츠가 스크린샷/영상 기반. 텍스트 피드백 희박
- 면접 연습 목적보다 **역량 측정 + 채용 연계 목적**에 최적화

### 강점 / 약점

**강점:** 대규모 기업 네트워크, 역량 기반 역전 매칭(기업이 먼저 제안), 낮은 진입 장벽
**약점:** 면접 코칭/연습 기능 없음. 피드백 미흡. 채용 플랫폼이 주, 연습 도구가 부

### MirAI 시사점

잡다는 채용 플랫폼. MirAI와 직접 경쟁 영역은 제한적. 단, **잡다 사용자(면접 앞둔 구직자)가 MirAI 핵심 타겟**이 될 수 있음.

## 2. 뷰인터 (ViewInter)

> 조사: 김관우 / B2B 공식 자료 + 기술 스펙 분석

### 기본 정보

- **운영사**: 제네시스랩 (Genesis Lab)
- **타겟**: B2B (기업 채용 담당자)
- **과금 모델**: 기업 단위 구독 (지원자 무료, 기업이 비용 지불)
- **특징**: 국내 AI 면접 선두주자 중 하나. 연구 기반 정량 평가

### 핵심 기술 스펙

| 측정 방식 | 세부 내용 |
|----------|----------|
| 안면 분석 | 68개 랜드마크 (눈·코·입·눈썹 등) |
| 시선 추적 | Eye tracking (좌우 이탈 감지) |
| 두부 움직임 | Head movement (고개 끄덕임·흔들림) |
| 음성 분석 | Energy (음성 강도), Pitch (음조) |
| 언어 분석 | STT (음성→텍스트), 충전어 감지, 키워드 매칭 |
| 데이터셋 | 50만 건 이상 면접 데이터 |
| 신뢰도 검증 | ICC 0.7~0.8 (인간 전문가와 AI 평가 일치도) |

### 8개 평가 지표

| 지표 | 내용 |
|------|------|
| 호감도 | 표정·시선·자세 종합 인상 |
| 신뢰도 | 시선 안정성, 답변 일관성 |
| 침착성 | 움직임 안정도, 표정 변화 진폭 |
| 활기 | 에너지 수준, 표정 활성도 |
| 유창성 | 발화 속도(WPM 130~150 기준), 충전어 비율 |
| 의사소통 | 논리 구조, 핵심어 포함률 |
| 적합도 | 직무 키워드 매칭률 |
| 전체 역량 | 종합 점수 |

### 강점 / 약점

**강점:**
- 정량적·재현 가능한 평가 (채용 편향 감소)
- 기업당 2,400시간+ 면접 절감 효과
- 50만 건 데이터 기반 검증된 모델
- 대기업·공공기관 채용 적용 레퍼런스

**약점:**
- 지원자 경험 최악: 카메라 + 마이크 고정. 30~60분 고압적 환경
- 피드백 없음: 결과를 지원자에게 공개하지 않음 (기업 전용)
- 맥락 파악 한계: 숫자 측정이지 내용 이해가 아님
- 기계적 대응 유도: "좋은 점수 받는 법" 학습이 오히려 자연스러움 훼손

### MirAI 시사점

뷰인터의 클라이언트는 기업. **지원자는 평가 대상일 뿐 서비스 수혜자가 아님.** MirAI는 지원자를 클라이언트로 보는 반대 접근. 뷰인터의 기술(멀티모달 분석)은 참고하되, 경험 철학은 정반대.

## 3. 몬스터 (Monster)

> 조사: 이승현 / 직접 사용 + 자료 조사

### 기본 정보

- **운영사**: 마이다스아이티 계열 (카피킬러 NLP 기술 활용)
- **타겟**: B2B (채용 담당자) — 지원자는 기업 요청 시 응시
- **핵심 포지션**: "AI로 거짓말 잡는 면접관"

### 핵심 기능

| 기능 | 내용 |
|------|------|
| 자소서 기반 팩트체크 | 자소서 내용 분석 → 일관성·진실성 검증 심층 질문 |
| 멀티모달 분석 | 표정 + 음성 + 시선 + 심박수(선택) |
| 방대한 데이터 | 실제 합격자 패턴 기반 평가 기준 |
| 치팅 방지 | 화면 이탈·시선 이탈·대리 응시 감지 |
| 직무 적합성 | 역량 자동 점수화 |

### 강점 / 약점

**강점:**
- 자소서 내용 기반 맞춤 팩트체크 질문 (단순 인성 질문 탈피)
- 강력한 부정행위 방지 시스템
- 방대한 합격자 패턴 데이터

**약점 (= MirAI 기회):**

| 약점 | MirAI 기회 |
|------|-----------|
| "평가" 위주 차가운 경험 — 연습 불가 | 연습·코칭 전문 서비스 |
| 수치 통보식 피드백 (이해 불가) | 실행형 피드백 ("어떻게 고칠 것인가") |
| 카메라+마이크 고정 환경 30~60분 | 유연한 환경, 짧은 세션 가능 |
| 결과 공유 없음 (기업 전용 리포트) | 지원자에게 투명한 결과 제공 |
| 맥락 이해의 한계 | RAG 기반 자소서 맞춤 질문 |

### MirAI 시사점

몬스터가 잘하는 것(자소서 기반 맞춤 질문, 팩트체크)은 MirAI 기능 1(자소서 기반 맞춤 질문 생성)과 겹침. 차별점은 **몬스터 = 기업을 위한 검증 도구, MirAI = 지원자를 위한 연습+코칭 도구**.

## 4. 사람인 AI 모의면접

> 조사: 멘토 / 웹 검색 (2026-03)

### 기본 정보

- **운영사**: 사람인 (국내 최대 구직 플랫폼)
- **출시**: 2025년 2월
- **타겟**: B2C 구직자 (사람인 회원)
- **가격**: 1회 10,000원 (정가 27,500원 → 63% 할인 중)

### 핵심 기능

| 기능 | 내용 |
|------|------|
| 맞춤 질문 | 이력서 + 채용 공고 기반 질문 자동 생성 |
| AI 면접관 페르소나 | 6명 AI 휴먼 면접관 (성격·스타일 다양) |
| 꼬리질문 | 답변 분석 후 심화 질문 생성 |
| 실시간 피드백 | 답변 중 즉각 피드백 |
| 결과 리포트 | 강점·약점 분석 + 예상질문 + 예시답안 제공 |
| 모바일 (2025.07+) | 음성 응시 기능 추가 |

### 강점 / 약점

**강점:**
- 사람인 회원 1,800만+ 기반 대규모 사용자 접근
- 이력서+채용공고 기반 맞춤 질문 (MirAI 기능 1과 유사)
- 6명 페르소나로 다양한 면접 스타일 제공
- 실시간 피드백 + 결과 리포트 존재

**약점:**
- 사람인 플랫폼 종속 (비가입자 접근 어려움)
- 1회 10,000원 — 반복 연습 시 비용 부담
- 자소서 PDF 업로드 기반이 아닌 이력서 텍스트 기반 (RAG 깊이 한계)
- 피드백이 "예시답안 제공" 수준 — 실행형 코칭 부재

### MirAI 시사점

**가장 직접적인 경쟁자.** 사람인 AI 모의면접이 이미 꼬리질문+피드백을 제공함. MirAI 차별점을 명확히 해야 함:

| 사람인 AI 모의면접 | MirAI |
|------------------|-------|
| 이력서 텍스트 기반 | PDF 자소서 전체 RAG 분석 |
| 1회 10,000원 | (미정 — 더 낮거나 구독) |
| 사람인 종속 | 독립 서비스 |
| 예시답안 제공 | 실행형 피드백 (무엇을 어떻게 고칠 것인가) |
| 6개 고정 페르소나 | HR·기술팀장·경영진 3종 맞춤 페르소나 |

## 6. MirAI 포지셔닝 인사이트

### 공백 지도

```
         B2B (기업 클라이언트)
              ↑
    뷰인터   몬스터
              |
평가·검증 ——————————— 연습·코칭
              |
    잡다           [MirAI 목표 위치]
              ↓
         B2C (구직자 클라이언트)
```

**MirAI 포지션:** B2C × 연습·코칭 × 자소서 RAG × 실행형 피드백

### 차별화 근거 (경쟁사 약점 기반)

1. **자소서 PDF → RAG 맞춤 질문**: 사람인(이력서 텍스트)·잡다(역량검사)가 못하는 자소서 깊이
2. **실행형 피드백**: 뷰인터·몬스터가 결과를 기업에만 제공, 지원자는 수치만. MirAI는 "어떻게 고칠 것인가"
3. **지원자 편 철학**: 기업 채용 도구가 아닌 지원자 성장 도구 (Equivalence 철학)
4. **3종 페르소나**: 사람인 6명 고정 vs MirAI HR·기술팀장·경영진 역할별 맞춤
5. **꼬리질문 유형화**: 기능 2의 CLARIFY·CHALLENGE·EXPLORE 3유형 — 사람인 미분화

### 주의사항

- 사람인 AI 모의면접이 이미 2025.02 출시, 기능 중복 — MVP 출시 전 차별점 더 명확히 필요
- 몬스터가 자소서 기반 질문을 하고 있으나 B2B(기업 전용). MirAI는 동일 기술을 B2C로
- 잡다의 40만+ 사용자는 MirAI 잠재 타겟. 잡다 이후 면접 준비 도구로 포지셔닝 가능
