## 핵심 원칙 (변경 불가)

1. **재미 1순위** — 모든 UX 결정에서 "재밌는가?"가 1번 기준
2. **코인 = 게임머니** — 현금화 없음. UI에서 원화 환산 표시 금지
3. **모바일 우선** — 375px 기준 설계, 데스크톱은 대응 수준
4. **브랜드 컬러** — 반드시 branding.md의 최종 확정 컬러 사용
5. **법적 의무** — dev_spec.md §보안·컴플라이언스 항상 준수
