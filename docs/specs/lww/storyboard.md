## 화면 1 [MVP]: 온보딩 — 스플래시 + 소개 슬라이드

### 시나리오

이취준이 앱스토어에서 Fint를 설치하고 처음 실행한다. 스플래시 화면이 1.5초간 표시된 후 3장의 소개 슬라이드가 나타난다. 슬라이드를 넘기면서 "면접 연습을 채팅으로 한다", "즉각적인 결과 리포트를 받는다"는 핵심 가치를 이해한다. 마지막 슬라이드에서 "바로 시작하기" 버튼을 눌러 로그인 없이 직군 선택으로 진입한다. **MVP에서는 로그인 없음 — 선 체험 후 가입. 크레딧·현직자 연결 기능 없음.**

### 목적

- 앱의 핵심 가치 3가지를 30초 안에 전달
- 로그인 없이 즉시 진입 — 마찰 제로, Aha Moment까지 최단 경로

### 핵심 요소

- 앱 로고 (보라 계열)
- 슬라이드 인디케이터 (● ○ ○)
- "바로 시작하기" CTA 버튼 (로그인 없음)
- 스킵 버튼 (우상단)

## 화면 1-2 [MVP]: 온보딩 — 프로필 설정 (취준생)

### 시나리오

로그인 없이 바로 진입한 이취준. "1단계만 하면 첫 면접을 시작할 수 있어요" 안내와 함께 직군 선택 + 취준 단계 선택이 1화면에서 통합 처리된다. 자소서 업로드는 Phase 1부터 제공. 완료 후 즉시 첫 면접 CTA가 표시된다.

### 목적

- AI 맞춤 질문 생성의 개인화 기반 마련
- 핵심 입력 1단계 원칙 준수: 직군 + 취준 단계를 1화면에서 동시 입력 → 즉시 면접 시작
- 마찰 최소화 — 로그인 없이 즉시 체험 진입

### 핵심 요소

- 직군 + 취준 단계 **1화면 통합** (스크롤 1페이지)
- 직군 선택 (최대 3개, 인기 직군 상단 고정)
- 취준 단계 선택 (서류 준비 / 면접 준비 / 최종합격 대기)
- 자소서 업로드 (선택, 스킵 버튼 제공, 업로드 시 혜택 강조)
- 첫 면접 시작 CTA

## 화면 3 [MVP]: 채팅형 모의면접

### 시나리오

이취준이 기본 제공 AI 면접관 중 "HR 김지수"를 선택하고 모의면접을 시작한다. 카카오톡처럼 친숙한 채팅 UI에서 AI 면접관이 질문을 던진다. 유저는 텍스트 또는 음성으로 답변한다. 면접 진행도를 상단 게이지로 확인할 수 있다. **MVP에서는 기본 AI 면접관 3종 제공 — 페르소나 마켓은 Phase 2+3 기능.**

### 목적

- 카카오톡 수준의 친숙한 UI로 면접 진입 장벽 제거
- 텍스트 입력 지원 (음성 입력은 Phase 1)
- 면접 진행률 시각화로 완료 동기 부여

### 핵심 요소

- 상단 바: 뒤로가기(← 탭 시 확인 팝업), 면접관 이름/상태, 진행도
  - **뒤로가기 정책 (MVP 확정)**: 확인 팝업 표시 — "면접을 중단할까요? 진행 내용은 저장되지 않습니다" [중단하기 / 계속하기]
- AI 메시지 버블 (왼쪽, 회색 배경)
- 유저 메시지 버블 (오른쪽, Primary 보라 배경)
- 하단 입력창 + 전송 버튼 (음성 버튼은 Phase 1)
- 타임스탬프

## 화면 4 [MVP]: 결과 리포트 + 합격 예언 오브

### 시나리오

5개 질문 답변 완료 (MVP 기준). 화면이 전환되며 결과 리포트가 등장한다 (Aha Moment). 종합 점수, 카테고리별 세부 점수, 실행 가능한 피드백을 확인한다. 하단에는 "합격 예언 오브" 예고 카드가 있어 Phase 1 후반 오픈 예정임을 안내한다. **MVP에서는 크레딧·결제 없음** — 오브는 잠금 상태로만 노출.

### 목적

- 면접 완료의 성취감 극대화 (Aha Moment)
- 합격 예언 오브 예고로 기대감 형성 (Phase 1 후반 오픈 예정)
- 재시도 유도 CTA

### 핵심 요소

- 카테고리별 점수 표 (논리력, 표현력, 자신감)

- 합격 예언 오브 예고 카드 (MVP: "곧 열려요" 배너, Phase 1 후반: 크레딧 소비 CTA)

## 화면 4-1 [Phase 1]: 로그인 / 가입 화면 (소셜 + 이메일)

### 시나리오

- 소셜 로그인 탭 (카카오·구글 버튼)

## 화면 9 [Phase 1]: 현직자 온보딩

### 시나리오

대기업 마케터 최현직이 "현직자로 가입"을 누른다. 역할 선택 → 소셜 로그인 → 회사 이메일 인증 → 페르소나 제작(AI 도우미 안내) → 첫 등록 완료 보상 순서로 20분 이내에 완료한다.

## 화면 10 [Phase 2+]: 직무 MBTI 카드 (크레딧 소비)

### 시나리오

모의면접을 5회 이상 완료한 이취준. 홈 피드 배너에서 "내 직무 MBTI 카드 생성됐어요!" 알림을 받는다. 크레딧을 소비해 자신의 면접 성향을 4가지 축으로 분석한 카드를 열람하고, SNS에 공유하여 바이럴을 발생시킨다.

### 목적

- Phase 2 핵심 바이럴 콘텐츠
- 크레딧 소비 동기 부여 (재미 + 자기 이해)
- SNS 공유로 신규 유입 촉진

### 핵심 요소

- 4가지 축 성향 바 (분석/실행, 협력/독립, 기획/운영, 언어/수치)
- 직무 유형 이름 + 한줄 설명
- 추천 직군 리스트
- SNS 공유 버튼 + 상세 보고서 크레딧 소비
