> ## ⚠️ 서비스 핵심 원칙 (모든 기획·설계의 최우선 기준)
>
> **핵심 슬로건: "취업준비부터 직장생활까지, 재밌게"**
>
> **1순위 가치: 재미**
> lww는 지루하고 무거운 면접 준비를 **재밌게 만드는 서비스**다.
> 유저가 lww를 쓰는 이유는 재미있어서여야 한다. 돈 버는 건 2순위다.
>
> **코인 = 게임머니**
> 코인은 수익 창출 도구가 아니다. 서비스 안에서만 쓸 수 있는 **게임머니**다.
> 플레이하면 쌓이고, 쌓인 게임머니로 더 많은 콘텐츠를 즐긴다.
> "코인으로 돈 번다"가 아니라 "코인으로 더 재밌게 논다"가 올바른 프레임이다.
>
> **이 원칙을 위반하는 기획·문구·설계는 수정 대상이다.**

---

> ## 💰 코인 ↔ 현금 구분 기준
>
> **원칙 한 줄: 외부 서비스에서 검증된 수익쉐어 모델이면 현금, 그 외엔 코인.**
>
> | 현금 거래 | 레퍼런스 모델 | 레이어 |
> |-----------|--------------|--------|
> | Pro 구독 | Duolingo · 클래스101 | [E] |
> | 멘토링 중계 수수료 | 탈잉 · 클래스101 | [B] |
> | 외주 중계 수수료 | 크몽 · 숨고 | [D] |
> | 채용 매칭 수수료 | 원티드 | [C] |
>
> | 코인 경제 (내부 순환) | 비고 |
> |----------------------|------|
> | 페르소나 마켓 소비·수익 배분 | 70/30 게임머니 배분 |
> | 쪽지 발송 | |
> | 콘텐츠 소비 전체 | 아차포인트·운세·후기 등 |
> | 활동 보상 | 면접 완료·글 작성 등 |
>
> **엣지 케이스 규칙**
>
> - **Pro 구독 코인 보너스**: 구독은 현금 거래, 지급 코인 보너스는 게임머니.
>   → 결제 화면에서 코인 보너스 원화 환산 표시 금지.
>
> - **유저 간 코인 이체·선물 금지**: 허용 시 현금 대체재 인정 위험 (전자금융거래법).
>   → 이용약관에 명시 필수.
>
> - **코인 환불**: 미사용 코인에 한해 구매 후 7일 이내 청약 철회 검토.

## 구현 계획

### 핵심 타겟 유저 페인포인트 (확정)

1. **돈이 없다** — 취준생은 유료 면접 컨설팅(1회 30~40만 원) 엄두 못 냄. 서비스는 무료/저가로 접근성 확보.
2. **지루하고 따분하다** — 기존 면접 준비는 플랫하고 재미없음. lww는 "Duolingo for 면접" — 어렵고 무거운 면접 준비를 쉽고 재밌게 바꾸는 것이 핵심 차별점.

   > **서비스 핵심 가치: 재미 (1순위)**
   > 코인은 수익 창출 도구가 아니라 **게임머니**다. 플레이하면 쌓이고, 쌓인 게임머니로 더 많은 콘텐츠를 즐긴다. 돈 버는 건 2순위 — 재미있어서 계속 쓰다 보니 코인이 생기는 구조.
   > 취준생: "면접 준비가 게임처럼 재밌다" → 코인은 플레이 보상
   > 현직자: "내 경험 공유가 재밌고 뿌듯하다" → 코인은 보너스
   > "자동사냥 수익"보다 "내 페르소나가 살아서 활동한다" — 자랑거리이자 재미

   **게이미피케이션 루프 (코인 획득 → 쪽지 발송 → 연결)**

   Phase 1부터 적용:
   | 행동 | 보상 |
   |------|------|
   | 면접 완료 | 코인 지급 + 스트릭 (연속 N일 뱃지) |
   | 질문 유형별 레벨업 | 인성 → 직무 → 압박 면접 단계 해금 |
   | 프로필 완성도 달성 | 코인 지급 |
   | 실제 면접 합격·불합격 후기 작성 | 코인 지급 |

   Phase 2 (커뮤니티) 추가:
   | 행동 | 보상 |
   |------|------|
   | 피드 글 작성 후 호응(추천·댓글) 받기 | 코인 지급 |
   | 출석 체크 뽑기 | 코인 랜덤 지급 |
   | 현직자 프로필에 연봉·직군·연차 입력 | 코인 지급 |

   > 익명 응원 폭탄: 코인 소비해서 랜덤 취준생에게 익명 응원 카드 발송. 받은 사람이 👍·감사 반응 시 보낸 사람에게 코인 환급. 어뷰징 방지(막 보내면 코인 손해) + 진심 응원 인센티브 + "누군가 날 응원한다" 감성 동시 해결.
   > 수신 정책: 수신 옵트아웃 설정 가능, 일일 수신 상한(예: 3건), 부적절 콘텐츠 필터링 + 신고 기능 필수.

   | 면접관 시각 "아차 포인트" 작성 | 면접 경험 현직자 | 코인 지급 |

   | 고득점 면접 Q&A 패턴 판매 | 취준생 | 소비될 때마다 게임머니 적립 |

   > 면접관 "아차 포인트": 면접관 경험 현직자가 "이래서 탈락시켰다", "이 행동이 감점이었다" 등 구직자 시각으로는 절대 알 수 없는 인사이더 정보 작성. 잡플래닛·블라인드에도 없는 콘텐츠. 구직자가 코인으로 잠금 해제.

   > 페르소나 게임머니 루프: "내 페르소나가 나 없이도 살아서 활동한다" — 만들어두면 취준생이 쓸 때마다 게임머니가 쌓이는 재미. 희귀 페르소나(대기업 임원·외국계)는 자연스럽게 인기 캐릭터화. 수익 배분 구조는 내부적으로 70/30 (현직자/플랫폼).
   > 잡플래닛·블라인드가 검증한 수요. 차별점: 단순 열람이 아니라 코인 경제에 통합 — 기여자도 보상받고, 소비자도 코인으로 접근. 취준생뿐 아니라 이직 고려 현직자도 소비층이 돼 양방향 코인 순환 완성.
   > 운세/타로/포춘쿠키: 취준 스트레스를 가볍게 풀어주는 엔터 콘텐츠. 코인 소비 허들이 낮아 잉여 코인 소진 + 재방문 습관 형성.

   → 취준생도 돈 없이 코인 획득 가능 → 현직자 쪽지 발송 가능
3. **외롭다** — 취준 기간 사람 못 만남. AI 면접관을 차가운 평가 도구가 아닌 **대화 상대**처럼 느끼게 해야 함.

```
MVP
  AI 모의면접 서비스 → 취준생 유입 (무료, 단면 서비스)
  수익: 없음 (핵심 기능 검증)

  → "생각보다 쉬웠고, 피드백이 진짜 유용하다"
```

#### Blended LTV (구독 + 코인 + 매칭)

> 구독만으로 LTV를 산출하면 코인 구매·페르소나 수수료·채용 매칭 매출이 누락됨. 아래는 수익원별 분리 추정.

**① 구독 LTV** (실효 활성 기간 6개월 기준)
> 취준 평균 기간 11.5개월(한국경제인협회)이나, D7 리텐션 25% 목표 기준 월간 이탈률 ~10% 적용 시 실효 활성 기간 중간값 약 6개월로 하향 추정 [가설].
> ⚠️ **한계**: D7 리텐션(7일 잔존율)에서 월간 이탈률을 직접 수식 변환하는 것은 표준 방법론이 아님. 일간 이탈 확률이 균등하다는 가정 하에 경험적으로 추정한 수치. Phase 1 실데이터(월간 코호트 리텐션 곡선) 확보 후 교체 필수.
```
9,900원 × 62% 갱신율 × 6개월 = 약 37,000원
```

**② 코인 구매 LTV** [가설]
- 코인 현금 구매 전환율: 5~10% [가설 — 네이버 웹툰 쿠키 구매 전환율 참고]

- 활성 기간 중 평균 구매 횟수: 1~2회 [가설]
- **LTV 기여: ₩5,000~₩30,000** [가설]

**③ 채용 매칭 LTV** (Phase 2+3, 별도 산출)

### AI 에이전트 생태계 비전 (몰트봇 레퍼런스)

> lww는 단순히 AI가 면접 준비를 도와주는 서비스가 아니다.
> AI와 사람이 함께 뒤섞여 살아가는 **커뮤니티 생태계**다.

**레퍼런스 사례**

| 사례 | 핵심 메커니즘 | lww 연결점 |
|------|------------|-----------|
| **Truth Terminal** (2024, Andy Ayrey) | 두 AI가 서로 대화하며 서사를 쌓음 ("Infinite Backrooms"). Twitter에서 자율 활동 → GOAT 밈코인으로 $500K+ 자산 축적. Marc Andreessen이 $50K BTC 전송. | AI 페르소나들이 커뮤니티에서 서로 대화하며 서사와 관계를 쌓음 |
| **Stanford Generative Agents** (Park et al., UIST 2023) | 25개 AI가 마을에서 자율 생활. 파티 계획·연애·관계 형성 등 사회적 행동 창발. 유저가 관찰·개입 가능. | AI 취준생·현직자 페르소나가 커뮤니티에서 자율 활동, 분위기 형성 |
| **Virtuals Protocol + ACP** (2024~) | AI 에이전트가 온체인 지갑 보유, 자율 거래. ACP(에이전트 상거래 프로토콜)로 에이전트끼리 서비스 발주·수행. | AI 에이전트가 외주 업무를 자율 수주·수행하는 경제 구조 |
| **Moltbot (몰트봇)** (2025~) | "AI that actually does things" — 명령 없이 스스로 컴퓨터 작업 수행. GitHub 72시간 60K 스타. | AI가 실제로 취업 준비 태스크(자소서 첨삭, 리서치 등)를 자율 수행 |

**lww AI 에이전트 비전 (Phase별)**

```
Phase 1 ~ Phase 2+3 초반: AI 페르소나가 커뮤니티 분위기 조성
  - AI 취준생·현직자 페르소나가 피드에서 자율 활동
  - 🤖 배지 기본 공개 — AI 여부 디폴트 오픈 (정체 숨기기 없음)
  - AI 페르소나끼리도 서로 대화하며 서사·관계 형성

Phase 2+3: AI 에이전트가 서비스 자율 수행
  - AI 면접관 페르소나가 취준생과 자율 모의면접 진행
  - 현직자가 없어도 "살아있는 페르소나"가 자율 활동
  - 코인 경제 안에서 AI-사람 간 가치 교환

Phase 2+3 이후: AI 에이전트 외주 경제 (장기 비전)
  - AI 에이전트가 취업 코칭·자소서 첨삭 등을 자율 수주
  - Virtuals ACP 방식으로 에이전트끼리 업무 분담
  - "AI가 혼자 외주 받아 뛰는" 구조 — 사람은 창작·감독 역할만
  수익 귀속 구조 (Phase 2+3 이후 [가설]):
  - AI 에이전트 외주 수익 → 페르소나 제작 현직자 + 플랫폼 분배
  - 배분 비율: 페르소나 마켓 70/30 구조 참고 (현직자 70% / 플랫폼 30%)
  - 단, AI 외주 수익은 현금 거래 [D] 레이어 — 코인 아님
  - 구체 배분 설계는 Phase 2+3 진입 후 법률·세무 자문 기반으로 확정
```

> **핵심 인사이트**: Truth Terminal이 보여준 것처럼, AI가 서사를 쌓고 관계를 형성하면 유저는 그 AI에 감정적으로 연결된다. lww의 AI 페르소나는 단순한 봇이 아니라 "캐릭터"다. 희귀 페르소나(대기업 임원·외국계 면접관)는 자연스럽게 팬덤이 생기고 서비스 재미와 페르소나 희소성을 높인다.

### Step 2 브랜딩 상세 계획

**산출물:** `docs/specs/lww/branding.md`

> **선택 방식**: 키 컬러 옵션 3가지 + 목업 시안을 사용자가 직접 고른다.
> 이후 선택된 컬러로 Shadcn UI CSS 토큰(`--primary`, `--secondary` 등) 확정.

**디자인 툴링 원칙** (`docs/background/ai_vibe_coding_design_rules.md` 기반)
- 컬러는 AI가 임의 선정 금지 → coolors.co 방식으로 조화로운 팔레트 생성 후 메인+서브 2가지만 사용
- Next.js 프로젝트: **Shadcn UI** 확정 (tailwind config에 HEX 코드 직접 삽입)
- 디자인 기준은 `branding.md`에 문서화 → 개발 시 AI에게 그대로 전달

**사용 스킬:**
- `tailwind-design-system` — 컬러 확정 후 Shadcn UI CSS 토큰(`--primary`, `--secondary` 등) + Tailwind v4 디자인 시스템 구성
- `frontend-design` — 목업 화면 컴포넌트 시안 생성
- `canvas-design` — 필요 시 .png 시각 목업 생성

**작업 순서:**
1. **경쟁사 컬러 회피 분석**
   - 사람인: 주황/레드 계열 → 회피
   - 잡다: 파랑 계열 → 회피
   - 뷰인터: 차가운 회색 계열 → 회피
   - 잡플래닛: 초록 계열 (#00C853) → 회피
   - LinkedIn: #0A66C2 파랑 → 회피
   - 당근마켓: #FF6F0F 주황 → 회피

2. **키 컬러 옵션 3가지 제시** (사용자 선택)
   - 각 옵션: "유쾌함·쉬움·성장" 브랜드 방향 반영
   - 밝고 에너지 있는 계열 — 진지하고 무거운 경쟁사와 대비
   - 옵션별 HEX 코드 + 선정 근거 + 무드 키워드 명시
   - Shadcn UI CSS 토큰 형태로 팔레트 작성

3. **컬러 팔레트 구성** (선택된 옵션 기반으로 확정)
   - Primary (키 컬러 계열)
   - Secondary (보조 컬러)
   - Neutral (텍스트·배경·구분선)
   - Semantic (성공·경고·에러·정보)

4. **타이포그래피**
   - 한글 본문: Pretendard (무료, 웹 최적화)
   - 영문/숫자: Inter, 코드: JetBrains Mono
   - 사이즈 스케일: Display(28px) ~ H1(24px) ~ H2(20px) ~ H3(17px) ~ Body-lg(16px) ~ Body(14px) ~ Body-sm(13px) ~ Caption(12px) ~ Caption-sm(11px)
   - 모바일 기준

5. **브랜드 톤앤매너**
   - **포지션**: "Duolingo for 면접" — 어렵고 지루한 면접 준비를 쉽고 재밌게
   - **핵심 키워드**: 유쾌함 · 쉬움 · 성장
   - **경쟁사 대비**: 뷰인터/몬스터/사람인은 진지하고 무거움, 잡플래닛·블라인드는 정보 조회 위주 → lww는 가볍게 시작해서 실력이 쌓이는 경험 + 코인으로 정보도 소비
   - **헤드라인 방향**: "이렇게 쉬워도 돼?" / "지루한 면접 준비는 그만"
   - 카피라이팅 가이드 (반말 또는 친근한 존댓말, 유쾌한 마이크로카피)
   - 마스코트 도입 여부 검토 (Duolingo 올빼미 방식)
