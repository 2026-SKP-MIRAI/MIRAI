# 2/14 멘토님 온라인 미팅 정리

## 1. 오늘 미팅의 큰 주제

AI를 잘 쓰려면 명세/기획서가 핵심이다. AI가 블랙박스가 되지 않으려면 기본 용어를 이해해야 한다.

## 2. 프로젝트 방향

페르소나 기반 면접관 만들기. 예: "김동현" 같은 특정 인물을 모델로 한 면접관.

필요한 데이터 3종:
- 유튜브 인터뷰 영상
- 강연/발표 영상
- 텍스트 콘텐츠 (블로그, 기사 등)

데이터 수집 계획을 먼저 세우는 것이 중요하다.

## 3. 작업 방식

AI와 합쳐서 일하는 방식으로 전환한다.

- "검색 → 플랜 짜기" 과정을 AI에 위임
- 컨텍스트 관리와 명세 품질이 결과물 품질을 결정
- Shift+Tab = Plan 모드 진입 (Claude Code)

## 4. 생각거리

아이작 아시모프 「최후의 질문」— SF에서 출발한 상상력이 기획으로, 기획이 현실 구현으로 이어진다.

## 5. 취업/커리어

- 가까운 미래는 변동성이 크다
- 니치 시장에서 시작해 큰 시장으로 확장하는 전략
- "바이브 코딩" — 빠른 프로토타이핑과 검증, 승산 있음

## 6. 과제/액션

- 경쟁사 기능 리스트업
- 기존 것에 한 숟갈 개선 추가
- 클론코딩 수준까지 구현해보기

## 7. 경쟁사 재정의

character.ai 같은 AI 챗봇 자체가 경쟁사다. AI는 특정 작업(A)만 하는 경향이 있어 틈새가 생긴다.
