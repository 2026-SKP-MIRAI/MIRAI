# claude-code-harness 적용 검토 계획

## 목표
https://github.com/chacha95/claude-code-harness 레포에서 MirAI에 적용할 수 있는 패턴·도구를 정리하고 실제 적용한다.

## 배경
- 이미 확인한 항목: exec-plans 파일 네이밍 규칙 (`-plan`, `-context`, `-tasks`) → 적용 완료
- 추가로 검토할 항목: CI 설정, 커스텀 Claude 명령, settings.json 패턴 등

## 단계별 작업
1. 레포 전체 구조 재검토 (최신 커밋 기준)
2. 적용 가능한 항목 리스트업
3. 팀 논의 후 적용 우선순위 결정
4. 항목별 실제 적용 및 문서화

## 완료 기준
- [ ] 적용 가능 항목 리스트 작성
- [ ] 팀 컨펌 후 우선순위 결정
- [ ] 선택한 항목 MirAI 레포에 반영

## 우선순위
Week 1~2 (개발 환경 셋업 단계)
